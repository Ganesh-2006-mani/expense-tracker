# Expense Tracker

A feature-rich Expense Tracker web application built using HTML, CSS, and JavaScript.
This app allows users to track income and expenses, categorize transactions, visualize spending patterns, and manage data efficiently.

---

## 🚀 Features

### 📊 Financial Tracking

* Add income and expense transactions
* Automatic balance calculation
* Separate income and expense totals

---

### 🏷️ Category System

* Assign categories (Food, Travel, Shopping, Bills, Salary)
* Category-based expense tracking
* Clear visibility of where money is spent

---

### 🖼️ Data Visualization

#### 🥧 Category-Based Pie Chart

* Shows spending distribution by category
* Updates dynamically when data changes

#### 📈 Monthly Line Chart

* Tracks spending trends over time
* Helps analyze monthly expense patterns

---

### ✏️ Edit Transaction (Important Feature)

* Modify existing transactions
* Update amount, description, or category
* Changes reflect instantly in UI and charts

---

### 🗑️ Delete Transaction

* Remove transactions easily
* Updates totals and charts instantly

---

### 💾 Data Persistence

* Uses browser localStorage
* Data remains even after page refresh

---

### 🎨 UI / UX

* Clean and minimal design
* Responsive layout
* Real-time updates
* Easy-to-use interface

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)
* Chart.js (for data visualization)

## ⚙️ How to Run

## 🎯 Core Functionalities

* Add, edit, and delete transactions
* Categorize expenses
* Track income vs expenses
* Visualize spending using charts
* Persist data using localStorage

---

## ⚠️ Limitations

* No user authentication
* No cloud/database storage
* No multi-device sync
* Limited category options (predefined only)
